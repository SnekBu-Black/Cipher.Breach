# CIPHER.BREACH

**CIPHER.BREACH** is a 3D browser-based cryptography game where the player escapes a hostile digital network by solving cipher challenges, avoiding daemon enemies, collecting the exit key, and clearing each room.

[Play the Game](https://snekbu-black.github.io/Cipher.Breach/)

![Main Menu](doc_assets/screenshot_01_main_menu.png)

## About the Game

Cipher Breach turns cybersecurity and cryptography concepts into a small playable experience. Each room challenges the player with a different cipher method while enemies, health, turns, and key objectives create pressure.

## Features

- 3D cyber-themed browser gameplay
- Arabic and English language support
- Desktop and mobile controls
- Enemy patrol and chase behavior
- Cipher battle system with health, timer, hints, and damage feedback
- Six main cipher rooms plus a hidden final challenge

## Cipher Rooms

| Room | Theme | Cipher |
|---|---|---|
| Room 1 | Cipher Relay | Caesar cipher |
| Room 2 | Token Cache | Reverse |
| Room 3 | Password Vault | Atbash |
| Room 4 | Identity Gate | Rail Fence |
| Room 5 | Payroll Gate | Scytale |
| Room 6 | Contractor Window | Columnar Transposition |
| Room 7 | Final Chamber | Secret |

## How to Play

Move through each room, avoid daemons, find the keyholder, solve cipher challenges, collect the exit key, and reach the exit node.
